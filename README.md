# CimaGo

A responsive web application built with the **Angular** framework that allows users to explore the latest movies and TV series. The application dynamically fetches real-time media data, including titles, posters, and ratings, using a third-party API.

## Features

- **Latest Content:** Dedicated sections for trending movies and TV series.
- **Dynamic Data Integration:** Fetches up-to-date pictures, names, and ratings via external API.
- **Component-Driven Architecture:** Built with reusable Angular components for a clean and scalable codebase.

## Tech Stack

- **Frontend:** Angular
- **Styles:** HTML5, CSS3 and Bootstrap
- **Data Source:** External REST API
